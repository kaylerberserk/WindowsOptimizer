## 📖 À propos du projet

**Windows Optimizer** est un script d'automatisation professionnel conçu pour transformer une installation Windows standard en une station de travail ou de jeu haute performance. 

### 🛡️ Stabilité & Polyvalence
Ce framework est **universel** : il a été rigoureusement testé et prouvé sur tous les environnements, du **PC de Gaming** au poste de **Bureautique**, en passant par les **Machines Virtuelles (VM)**. Contrairement aux versions "Lite" ou modifiées de Windows, ce script **ne supprime aucune fonctionnalité** système. Vous gardez un Windows complet, mais parfaitement optimisé.

## 🚀 Démarrage Rapide (Moins de 5 minutes)

1. **Téléchargement** : Accédez à la page du fichier [**All in One.cmd**](https://github.com/kaylerberserk/WindowsOptimizer/blob/main/All%20in%20One.cmd) et cliquez sur le bouton **Download**.
2. **Exécution** : Clic droit sur le fichier → **Exécuter en tant qu'administrateur**.
3. **Sécurité** : Appuyez sur **[R]** pour créer un point de restauration avant de commencer.
4. **Optimisation** : Choisissez le profil qui vous correspond (**[D]** pour Bureau, **[L]** pour Portable).
5. **Redémarrage** : Redémarrez votre PC pour finaliser les changements.

### ⚙️ Optimisations Granulaires
*Si vous préférez choisir précisément ce que vous améliorez :*

- **[1] Système** : Rend Windows beaucoup plus réactif et fluide au quotidien.
- **[2] Mémoire** : Optimise votre RAM pour éliminer les saccades dans les jeux.
- **[3] Disques** : Accélère le chargement de vos jeux et logiciels.
- **[4] GPU** : Réduit le délai d'affichage et améliore la fluidité visuelle.
- **[5] Réseau** : Réduit votre ping et stabilise votre connexion internet.
- **[6] Input** : Rend votre souris et votre clavier plus précis et instantanés.
- **[7] Énergie** : Débloque la puissance maximale de votre processeur.
- **[8] Sécurité** : Gagnez quelques FPS en ajustant les protections processeur (avancé).

### 📂 Gestion Windows (Menu [G])
*Pour reprendre le contrôle sur les composants imposés par Microsoft :*

- **Windows Defender** : Gardez l'antivirus actif ou coupez-le pour libérer de la puissance.
- **IA & Widgets** : Supprimez Copilot, Recall et les widgets pour un Windows plus propre.
- **Animations** : Choisissez entre une interface "jolie" ou une interface ultra-rapide.
- **Désinstalleurs** : Supprime proprement OneDrive et Edge si vous ne les utilisez pas.
- **Runtimes** : Installe le nécessaire pour que vos jeux ne plantent pas au lancement.

## ❓ FAQ (Foire Aux Questions)

### 🏠 Installation & Sécurité

**Q : Est-ce que ce script va "casser" mon Windows ?**  
**R :** Non. Il utilise des méthodes documentées. Contrairement à certains outils qui suppriment des fichiers vitaux, ce script ajuste uniquement des réglages. L'option **[R]** vous permet de revenir en arrière instantanément.

**Q : Mon antivirus détecte le script comme une menace, pourquoi ?**  
**R :** C'est un "faux positif". Le script modifie des réglages système, ce qui est jugé suspect par certains antivirus, bien que les actions soient ici bénéfiques.

**Q : Puis-je utiliser ce script sur Windows 10 et 11 ?**  
**R :** Oui. Le script détecte votre version et applique les réglages spécifiques (IA, gestion des cœurs processeurs récents, etc.).

### 🎮 Performance & Gaming

**Q : Quel gain de FPS puis-je espérer ?**  
**R :** Cela dépend de votre PC. Vous ressentirez surtout une meilleure stabilité (moins de saccades) et une réponse plus rapide de votre souris et clavier.

**Q : Est-ce que je risque un bannissement dans mes jeux ?**  
**R :** Absolument pas. Le script n'injecte rien dans les fichiers de jeux. Il optimise simplement la manière dont Windows gère votre PC.

### 🌐 Réseau & Internet

**Q : Est-ce que mon ping va vraiment baisser ?**  
**R :** Oui, surtout si vous jouez en ligne. Le script donne la priorité aux données de vos jeux sur votre connexion internet.

**Q : Puis-je toujours mettre à jour mon PC avec Windows Update ?**  
**R :** Oui. Le script ne bloque pas les mises à jour. Il nettoie simplement les anciens fichiers inutiles pour gagner de la place.

### 💻 Maintenance & Divers

**Q : Puis-je réinstaller OneDrive ou Edge plus tard ?**  
**R :** Oui, vous pouvez les retélécharger sur le site de Microsoft à tout moment.

**Q : Est-ce que ça fonctionne sur les PC portables ?**  
**R :** Oui, utilisez le profil **[L]**. Il booste les performances tout en faisant attention à votre batterie.
